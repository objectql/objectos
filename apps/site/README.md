# ObjectOS Documentation Site

The official documentation and marketing website for ObjectOS, built with Next.js and Fumadocs.

## Features

- **Documentation Engine**: Powered by Fumadocs for beautiful, accessible MDX-based docs.
- **Blog**: Built-in blogging capabilities.
- **Tailwind CSS**: Styled with Tailwind for modern design.
- **Interactive**: Examples and interactive components.

## Development

```bash
# Start development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## Structure

- `/app`: Next.js App Router pages.
- `/content/docs`: Documentation MDX files.
- `/content/blog`: Blog post MDX files.
