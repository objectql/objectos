# DataTable Filter Examples

This directory contains example code demonstrating how to use the enhanced filter functionality in the ObjectQL DataTable component.

## Files

- **filter-examples.tsx** - Comprehensive examples showing different filter usage patterns

## Examples Included

### 1. Single Column Filter (Legacy)
The simplest way to add filtering - filter on a single column with a search input.

### 2. Multiple Column Filters
Display multiple filter inputs in a responsive grid layout, allowing users to filter by multiple columns simultaneously.

### 3. Multiple Filters (No Count Display)
Use multiple filters but hide the active filter count indicator.

### 4. Custom Filter UI
Build a completely custom filter interface using the standalone `DataTableFilter` component.

### 5. Filters Toolbar
Use the `DataTableFiltersToolbar` component for a horizontal filter layout.

## Running Examples

These examples are provided as reference code. To use them in your application:

1. Copy the example code you need
2. Adjust the data types and columns to match your use case
3. Import the necessary components from `@objectql/ui`
4. Customize styling and behavior as needed

## Key Components

- **DataTable** - Main table component with built-in filter support
- **DataTableFilter** - Standalone filter input for a single column
- **DataTableFiltersToolbar** - Pre-built toolbar for multiple filters

For more details, see `FILTER_USAGE.md` in the parent directory.
